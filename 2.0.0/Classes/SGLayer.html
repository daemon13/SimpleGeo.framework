Overview

An SGLayer object represents a SimpleGeo Storage layer. Storage layers hold SGStoredRecord objects. Each Storage layer has a unique name. SGLayer objects also store information about when the later was created and last updated. An array of callback URLs may be attached to a layer; these URLs are notified every time a record is added or updated.
