Overview

An SGAddress object stores address data. Address data includes information such as country, state, city, and street address. SGAddress objects are typically generated by a Context response or stored as part of an SGPlace object, although you can also create them explicitly yourself.

  • Create an SGAddress from an NSDictionary containing standard address keys (city, province, etc).
  • Generate a a formatted address string from an SGAddress using [SGAddress formattedAddress:withStreet:].
