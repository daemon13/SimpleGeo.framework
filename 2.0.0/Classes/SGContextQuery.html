Overview

An SGContextQuery object stores query information for a SimpleGeo Context API request. To make a Context request, create an SGContextQuery object and call [SimpleGeo getContextForQuery:callback:].

  • Specify filters to request only specific parts of a full Context response.
  • Specify feature categories to limit the types of polygons returned in the response.
