Overview

A SimpleGeo object acts as a client for all API requests. A SimpleGeo object is created with SimpleGeo credentials (an OAuth consumer key and secret). All API requests are made by calling methods in this class.

// Create the client
SimpleGeo *client = [SimpleGeo clientWithConsumerKey:@"key"
                                      consumerSecret:@"secret"];
// Example request
[client getContextForQuery:query callback:callback];
