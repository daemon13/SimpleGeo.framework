Inherits from SGGeometry : NSObject
Conforms to SGRegionGeometry
Declared in SGPolygon.h
SGPolygon.m

Overview

An SGPolygon object stores information about a polygon. Point information is stored in an array of rings. The first ring of points represents the polygon’s boundary. Additional rings contain information about holes in the polygon.
