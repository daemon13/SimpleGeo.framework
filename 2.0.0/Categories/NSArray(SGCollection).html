Overview

This category on NSArray allows an NSArray to be created from a GeoJSON Collection. It also allows a GeoJSON Collection to be created from an NSArray. Call arrayWithSGCollection: to transform a GeoJSON Collection of SimpleGeo places or records into an array of SGPlaces or SGStoredRecords.
