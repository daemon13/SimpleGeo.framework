Overview

SGNearbyQuery serves as an abstract class for a nearby API request. With an SGNearby query, you may:

  • Constrain results with a radius
  • Limit the number of results with a limit

Important: You should never create an SGNearbyQuery directly, but instead create an SGContextQuery or SGStorageQuery object (both are subclasses of SGNearbyQuery).
