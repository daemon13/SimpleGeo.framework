Overview

An SGStoredRecord object represents a record in a SimpleGeo Storage layer. Like any SGGeoObject, a SGStoredRecord object stores a geometry (specifically, a point) and a properties dictionary. A SGStoredRecord object also stores the name of its associated layer and a creation date. If the SGStoredRecord originated from an API query response, it will also store a distance value (its distance from the query point).
